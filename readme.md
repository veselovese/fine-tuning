Fine-Tuning

Обзор проекта
Проект посвящен решению задачи классификации изображений с использованием методов Transfer Learning. В рамках работы собран собственный датасет, проведены эксперименты с моделями семейств ResNet и EfficientNet, выполнен подбор гиперпараметров и осуществлен экспорт лучшей модели в формат ONNX для инференса на CPU.

Структура проекта
data/ — содержит сырые данные и описание датасета.
experiments/ — Jupyter Notebook с исследовательским анализом и экспериментами.
fine_tuning/ — скрипт train.py для финального обучения модели и экспорта в ONNX.
web_app/ — приложение на базе Gradio для инференса.
src/ — вспомогательные утилиты.

Настройка окружения
Для работы необходим Python 3.8+.Установите зависимости:
```
pip install -r requirements.txt
```

Данные
Датасет собран вручную и содержит 3 класса: plates, mugs, tableware. Всего не менее 90 изображений (минимум 30 на класс).Подробное описание метода сбора и определений классов находится в data/README.md.

Обучение и Эксперименты
Запуск экспериментов
Для анализа и подбора гиперпараметров запустите ноутбук:
```
jupyter notebook experiments/analysis.ipynb
```

Финальное обучение
Для обучения лучшей модели и экспорта в ONNX выполните:
```
cd fine_tuning
py train.py --model resnet18 --epochs 15
```
Результат (.onnx файл) будет сохранен в fine_tuning/models/.

Запуск веб-приложения
Для запуска локального приложения классификации:
```
cd web_app
py app.py
```
Приложение будет доступно по адресу http://127.0.0.1:7860.
