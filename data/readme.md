Описание набора данных

Метод сбора
Изображения были собраны вручную c использованием личного архива. Отбирались четкие фотографии объектов на нейтральном фоне для облегчения процесса обучения.

Классы
Датасет состоит из 3 классов:
Plates (Тарелки) - плоская посуда для подачи пищи. Включает глубокие и мелкие тарелки.
Mugs (Кружки) - сосуды с ручкой для питья горячих напитков.
Tableware (Столовые приборы) - прочие предметы сервировки. Включает ложки, вилки, чайные ложки.

Задача
Мультиклассовая классификация изображения на один из трех классов посуды.

Предварительная обработка
Изменение размера (Resize) до 224x224 пикселей.
Нормализация по средним и стандартным отклонениям ImageNet.
Аугментация для обучающей выборки: случайный поворот, горизонтальный переворот, изменение яркости.
